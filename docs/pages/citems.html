Permissions

Permission linked Command Usage
citem.save /citem save Save an item to the database as a Citem
citem.delete /citem delete Delete a Citem from the database
citem.update /citem update Updates a Citem
citem.modify.rename /citem rename Rename an item
citem.modify.lore /citem lore Modification of an Items lore
citem.modify.flags /citem flag Adds a flag to an Item
citem.modify.actions /citem action Adds an interaction to a citem
citem.give /cgive Give a Citem to a player
citem.remove /cremove Removes a Citem from a player
citem.tag /citem tag Adds Tags to an Item
citem.info /citem info Displays the information of a Citem
citem.sign /sign Sign an Item

Commands

Citem rename

Syntax: /citem rename [name]

NOTE: You have to hold an item

Through the /citem rename Command you are able to rename an Item to a name of your choosing. This supports default minecraft, but also HEX color codes.

Color code Type Example Command Output
Hex Code /citem rename &x&A&B&9&9&D&EJohnny Name of the item:

Johnny
Default Minecraft /citem rename &6Johnny Name of the item:

Johnny

How to create a Citem

This is a walkthrough on how to create a Citem!

Steps Commands Explanation Example
1 Get an Item! Get an item of your choosing to make a Citem with. Pufferfish
2 /citem rename [Name of your item] Give your future Citem a Name(You can use color codes) /citem rename &x&A&B&9&9&D&EJohnny
3 /citem lore add Add some lore to your Citem, it can be anything! don't forget to add an empty line at the end. /citem lore add &7The Debug Pufferfish!
/citem lore add &7
4 /citem tag Add fitting tags to your item. You can check out the available tags here: Tags /citem tag Collectable Mythic
5 /citem flag To add protection to the citem you should always add flags to it. /citem flag add Undroppable
/citem flag add Unusable
/citem flag add Unbreakable
6 /citem action If you want your citem to execute an interaction, you can add it like that. /citem action left johnny_left
/citem action right johnny_right
7 /citem save Last but not least: Save your citem under an ID so it gets picked up by the database! /citem save johnny
- /cgive You can now give a player(or yourself) your newly made Citem! /cgive HektorTM johnny
